• Mouse: Un mouse es un dispositivo manual utilizado para interactuar con una computadora. Se trata de un periférico de entrada imprescindible en un ordenador de escritorio para la mayoría de las personas. Funciona detectando el movimiento que realiza el usuario al desplazarlo sobre una superficie plana, traduciendo ese movimiento a la pantalla a través de un cursor o puntero, usualmente en forma de flecha.
  • Teclado: Un teclado es un dispositivo de entrada que se utiliza para introducir datos en una computadora, como texto, comandos y funciones.
  • Monitor: Un monitor es un dispositivo de salida visual que muestra información de una computadora. Piensa en él como la televisión de tu computadora, ya que te permite ver las imágenes, textos y gráficos generados por el ordenador.
